Memory2D

Memory is the classic card puzzle game where the player has to pick two cards with the same image. the idea is to remember what picture is behind each card its a simple game but is engaging enough to play for hours!

Basic Gameplay

  • There are 12 cards
  • Each card has an image at the back
  • 2 cards have the same image
  • The images are randomly placed
  • User has to click on each card to find the image behind
  • User has to consecutively choose two cards with same image
  • time and turns are calculated and shown when all cards are found
