HangMan2D

Hangman is a simple word guessing game where the player has to guess the word before they run out of chances Each wrong letter will contribute to the completion of a hanging man picture and once the picture is completed the player lose If the player guesses the word before the picture is completed , They win.

Basic Gameplay

  • A text containing a list of words are read into the program.
  • Each time a random word is selected and assigned to a string answer
  • The text from Button behavior is compared with answer letter by letter
  • An image array is made with all the body parts of hangman
  • If present it is set as the text of the blank line corresponding to its position else part of image is displayed
  • If all elements of image are displayed The player failed to guess
  • if all the letters are displayed in blank position the score of player is incremented
