Battle-Ship

The Game is coded in C++ and played in the out put console. Game is made with minimum UI just to focus on the game play mechanics.

Basic Gameplay

  • There are 2 Battleships and 4 destroyer hidden in 10x10 matrix boxes
  • BattleShips are of size 5(boxes) and destroyers are of size 2(boxes)
  • User has to choose a row and coloumn to select the box
  • If ship 'S' is displayed, if Destroyer 'D', if empty 'O'
  • if all the ships and destroyers are found usr wins
  • User can quit in the middle and then the original location of the ships are shown
  • Git source : https://github.com/GRaj-gitHub/BattleShip_game.git
